# 🏀 Michael Jordan — Chicago Bulls Seasons (1984–1998)

A clean, static website showcasing every Chicago Bulls season Michael Jordan played, including:
- Season-by-season stats  
- Team records  
- Shooting splits  
- Playoff outcomes  
- Championship seasons highlighted  

This project is built as a lightweight HTML/CSS/JS site that loads season data from a CSV file.  
Perfect for GitHub Pages hosting.
